# 魚魚呑 SNS戦略（たたき台）

## 戦略の根幹
**「グルメサイト経由予約 → SNS経由の自社予約」へシフト**
食べログ・ホットペッパーの掲載料を中長期で削減し、ファン化された顧客から直接予約が入る状態をつくる。

## ファネル設計
```
[認知]   Instagram Reels / TikTok / Googleマップ
   ↓
[興味]   フィード投稿 / 口コミ / 内装・料理写真
   ↓
[来店動機] ストーリーズ・限定情報・本日のおすすめ
   ↓
[予約]   DM予約 / LINE予約 / Googleマップ予約 / 自社予約フォーム
   ↓
[再来店] LINE公式でのリピート促進・限定特典
```

## プラットフォーム別方針

### Instagram（主軸）
- **Reels（週2-3本目安）:** 看板料理の調理シーン・仕入れ風景・店主の人柄
- **フィード（週2-3本目安）:** メニュー写真・季節の魚紹介・店内雰囲気
- **ストーリーズ（毎日）:** 本日のおすすめ・空席情報・スタッフの裏側
- **プロフィール導線:** 「DM予約OK」「LINE予約はこちら」を明示
- **ハッシュタグ:** #大田区グルメ #蒲田居酒屋（エリア要確認）#海鮮居酒屋 等

### TikTok
- **Reelsと同素材を縦動画用に最適化**して横展開
- 食欲喚起（しずる感）コンテンツを優先
- ローカルハッシュタグ + 海鮮系トレンドを併用

### Googleマップ / MEO
- ビジネスプロフィール最適化（写真・営業時間・メニュー・属性）
- 口コミ獲得の仕組み化（来店客への口コミ依頼カード等）
- 投稿機能の活用（週1更新目安）
- 予約ボタンの導線設置

## 自社予約導線の整備（最優先タスク）
予約導線が整っていないとSNSで集客しても結局グルメサイトに流れる。先にここを固める。
- [ ] LINE公式アカウント開設 or 整備
- [ ] 自社予約フォーム（オウンドサイト or Googleフォーム）
- [ ] Instagram予約ステッカー設定
- [ ] Googleマップ予約導線

## 月次施策スケジュール（想定）
- **1ヶ月目:** ヒアリング・現状分析・アカウント整備・予約導線設計・撮影開始
- **2-3ヶ月目:** 投稿運用本格化・フォロワー獲得・口コミ獲得施策
- **4-6ヶ月目:** データを見ながら勝ち筋にリソース集中・自社予約比率を計測

## KPI候補
- 自社予約数（月次・前月比）
- Instagramフォロワー数
- Reels再生回数 / 保存数
- Googleマップ口コミ数 / 平均★
- 食べログ・ホットペッパー予約数（減少を確認）
