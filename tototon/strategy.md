# 魚魚呑 SNS戦略（2ヶ月モニター版）

## 戦略の根幹
**「短期勝負・20時以降の自社予約に効く動線」だけに集中する。**

モニター契約2ヶ月という制約から、ブランディング・フォロワー育成のような中長期施策は2ヶ月目以降に回し、
1ヶ月目は**「予約に直結する動線整備」と「20時以降需要を刈り取る発信」**に絞る。

## ターゲット仮説
- **メインターゲット:** 蒲田駅徒歩圏で働く20代後半〜40代のサラリーマン・OL
- **来店動機:**
  - 一次会（仕事終わり19時前後）
  - **二次会・サク飲み（20時以降）← ここを取りに行く**
  - 飲み会後の「もう一軒だけ」「シメに刺身と日本酒」
- **強み仮説:** 海鮮の鮮度＆客単価4,000-5,000円のコスパ感

## プラットフォーム別方針

### Instagram（主軸・既存資産活用）
**前提：投稿235件あるのにフォロワー1,000以下＝コンテンツ方針が刺さっていない。リブランディング寄りで再起動。**

- **プロフィール改修:**
  - 一行目に「蒲田駅2分・海鮮居酒屋」「20時以降のサク飲み歓迎」
  - 予約導線（DM / LINE / GMAP）を明示
- **Reels（週2本）:** 看板料理の調理・仕入れの新鮮さ・店主の人柄
- **フィード（週1-2本）:** メニュー単品 × しずる感
- **ストーリーズ（毎日・特に夕方〜夜）:**
  - **平日19時台 / 20時台に「今日の空席状況」リアルタイム発信** ← 最大の差別化策
  - 二次会需要への直接訴求
- **ハイライト:** 「メニュー」「予約方法」「アクセス」「20時以降」

### TikTok（新規開設・実験枠）
- Reelsと同素材を縦動画用に最適化して横展開
- 蒲田・大田区ローカルタグ + 海鮮居酒屋トレンドタグ併用
- **#蒲田グルメ #蒲田居酒屋 #蒲田二次会** などローカル軸

### Googleマップ / MEO（隠れた主役）
**地理的に蒲田駅2分は強烈なアドバンテージ。MEOで「蒲田 居酒屋」「蒲田 海鮮」上位が取れたら自社予約直結。**

- ビジネスプロフィール最適化（写真・カテゴリ・属性・営業時間）
- 投稿機能を週1更新（Instagram投稿の流用OK）
- **口コミ獲得施策:**
  - 来店客への口コミ依頼カード設置
  - 「口コミ書いてくれた方にサービス」スキーム検討
- 予約ボタンの動線設置

## 自社予約導線の整備（Week 1-2で完了マスト）
予約導線が整っていないと、SNSで集客しても結局グルメサイト経由になる。**先にここを固める。**

### 検討中の構成案
- **A案:** Instagram DM予約 + Googleマップ予約のみ（最軽量）
- **B案:** LINE公式アカウント開設 + Instagram/TikTok/GMAP全媒体からLINEへ集約（リピート促進も狙える）
- **C案:** 簡易予約フォーム（Googleフォーム / STORES予約）+ 各SNSからリンク

→ **推奨はB案**（リピート促進まで含められる・モニター期間後も資産として残る）

## 20時以降需要の刈り取り施策（独自性）
他のSNSコンサルがやらない領域。これを差別化として全面に出す。
- 平日19:30-20:00にストーリーズで「本日の空席速報」
- 「電車一本のサラリーマン向け」訴求コピー
- 「20時以降ご来店で◯◯サービス」など仕掛けの提案（要オーナー合意）
